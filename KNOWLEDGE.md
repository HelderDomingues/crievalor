
# Project Knowledge Base

## Project Overview
- **Project Name**: Crie Valor Estrat√©gia
- **Description**: A website for Crie Valor, showcasing their MAR (Mapa para Alto Rendimento) solution that combines human consulting with artificial intelligence for creating exceptional business strategies.
- **Primary Goal**: To inform visitors about the company's services and convert them into clients.

## Design Guidelines

### Color Scheme
- Primary: Blue shade (hsl(217, 91%, 60%))
- Background: Dark blue/navy (hsl(220, 35%, 9%))
- Accent colors: Purples and light blues for AI-related visuals

### Typography
- Font Family: Inter (primary font)
- Headings: Tracking tight, bold weights
- Body text: Regular weight, good readability

### Visual Style
- Maritime theme with AI elements
- Animated grid mesh backgrounds
- Wave animations
- Dark mode by default with high contrast elements

## Page Structure
1. **Home page (Index)**: Main landing page with hero section, services overview
2. **MAR page**: Details about the MAR methodology
3. **About page (Sobre)**: Company information and values
4. **Contact page (Contato)**: Contact forms and information
5. **Legal pages**: Privacy Policy, Refund Policy, Terms of Service

## Key Components
- HeroSection: Main banner with animations
- MaritimeWaves: Wave animation for maritime theme
- AIGridMesh: AI-themed grid background animation
- FeatureCards: Display service features
- ContactSection: Contact form and information
- Chatbot: Interactive chat assistant

## Technical Stack
- React + TypeScript
- Vite as the build tool
- Tailwind CSS for styling
- shadcn/ui for UI components
- React Router for navigation
- React Query for data fetching
